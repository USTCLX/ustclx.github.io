常用React hooks

2021-04-05 react 0 评论 字数统计: 213(字) 阅读时长: 1(分)

React hooks 已经很普及了。记录和整理一些比较通用的 hook。

useMount

返回组件挂载的状态。可以在一些异步处理之后，判断组件是否已经被卸载。

usePreProps

返回上一次的 props，也可以用来判断某个 props 是否改变。

useOnScreen

返回组件是否曝光。
